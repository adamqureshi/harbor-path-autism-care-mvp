# Deployment Checklist

## GitHub

1. Create repo: `harbor-path-autism-care-mvp`
2. Upload all files from this folder.
3. Commit to `main`.

## Vercel

1. Add New Project.
2. Import the GitHub repo.
3. Framework preset: Other / Static.
4. No build command needed.
5. Deploy.

## Custom domain

1. Buy a domain.
2. In Vercel, open Project Settings → Domains.
3. Add the domain.
4. Copy the DNS records Vercel gives you.
5. Add those records at your domain registrar.
6. Wait for verification and SSL.

## Before sharing publicly

- Replace `hello@harborpathcare.com` in `script.js` with your real email.
- Add Google Analytics, Vercel Analytics, or Plausible.
- Test both forms.
- Test mobile layout.
- Review all clinical claims with your wife and counsel.
- Remove any statement that implies diagnosis, guaranteed coverage, or AI-only therapy.
